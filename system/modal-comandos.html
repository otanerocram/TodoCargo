Comandos de Voz para TodoCargo

Instrucciones
Click en "Permitir" para acceder al micrófono y activar los comandos de voz. Verás un ícono de micrófono AZUL parpadear indicando la activación. Verás el ícono de color ROJO si "No Permites" el acceso o si no tienes ningún micrófono instalado.

Como medida de seguridad, tu navegador desconectará el micrófono cada 60-120 segundos si ningún comando es recibido. En ese caso se solicitará "Permitir" or "Denegar" acceso al micrófono.

Commandos
  • 'Clientes' - Muestra la lista de Clientes
  • 'Recibir Carga' - Abre ventana de recepción de carga
  • 'Despachar Carga' - Registrar una entrega de carga
  • 'Carga Entregada' - Ver/Buscar carga entregada
  • 'Facturas' - Abrir Facturas
  • 'Boletas' - Abrir Boletas
  • 'Desactivar Sonidos' - Silencia todos los sonidos del sistema.
  • 'Activar Sonidos' - Activa todos los sonidos del sistema.
  • 'Desactiva comandos de voz' - Desactiva los comandos de Voz.
  • 'Ayuda' - Muestra la lista de comandos
  • 'OK' - Cierra esta ventana
  • 'Ocultar Menú' - Oculta la barra de navegación.
  • 'Mostrar Menú' - Muestra la barra de navegación.
  • 'Arriba' - Sube la página.
  • 'Abajo' - Baja la página
  • 'Regresar' - Vuelve hacia la página anterior
  • 'Salir' - Cierra la sesión del sistema
